### 方案 A：最專業的方法 — 啟動 GitHub Pages (推薦 🌟)
這是將您的 GitHub 專案變成一個真實網站的標準做法。

進入設定：在您的 GitHub 頁面上方，點擊 「Settings」 (齒輪圖標) ⚙️。

尋找 Pages：在左側選單中，找到 「Pages」 選項。

配置來源：

在 Build and deployment 下方的 Branch，選擇 main。

右側資料夾選擇 /(root)。

點擊 Save。

等待交屋：大約等待 1-2 分鐘，頁面頂端會出現一條橫幅，顯示："Your site is live at ..."。

點擊網址：點擊該網址，並在網址後面加上 /VibeCoding_Calculator.html，即可看到結果！

### 方案 B：最快速的方法 — 使用 HTML Preview (秒看結果 ⚡)
如果您只是想快速測試，不需要正式網址：

複製您的 GitHub 文件網址：https://github.com/Grace-TA/IS2025/blob/main/VibeCoding_Calculator.html

在網址前面加上 htmlpreview.github.io/?

變成：https://htmlpreview.github.io/?https://github.com/Grace-TA/IS2025/blob/main/VibeCoding_Calculator.html

直接按下 Enter，網頁就會「浮現」出來！
